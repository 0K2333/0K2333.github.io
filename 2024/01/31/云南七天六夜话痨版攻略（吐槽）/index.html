云南七天六夜话痨版攻略（吐槽）

发表于2024-01-31|更新于2024-01-31|随笔山川河流
|字数总计:5612|阅读时长:16分钟|阅读量:|评论数:

——昆明1 -> 大理2 -> 丽江2 (4.21-4.27 七天六晚)

Day0—4.21 出发✈

  • 计划：

    1. 南京-贵阳 🛫 （禄口机场-龙洞堡机场） + 10kg行李托运 17：30-20：05 491r
    2. 打车到贵阳站附近的二七路小吃街，吃晚饭！
    3. 贵阳-昆明 🚆 （贵阳站-昆明站） 火车软卧 22：07-6：21 244r

  • 实际：

    1. 飞机延误，19：10分出发，21：50到达贵阳…由于贵阳机场太大，出机口离取行李的地方太远，最终没能赶上火车，申请了退火车票，扣了180r的手续费😭
    2. 时间太晚夜市也没心情去了，不知道好不好吃。临时定了贵阳高铁站附近的如家精选酒店暂住一晚，买了22号早上7：46-10：21（贵阳北站-昆明南站）的高铁票 264r/3人+212.5r

  • 感受：

    1. 虽然说飞机延误是常态，但是大学生们还是不要为了穷游省钱选择我们这种麻烦的中转，看似钱少了，但是省下来的钱真的会在后续加倍补上去啊🥲！没有什么比直飞更方便的了！！果然该花的钱还是得花…
    2. 飞机还是尽量选择有名的航班吧，这趟的飞机选的是九元航空，延误就算了，他还不提供吃食！而且延误了之后！不可以免费退票！！改签只能选择当天的九元航班改签！而那天九元航空，就我们延误的这趟！！！心疼改签费我们硬着头皮坐了这趟延误的飞机！飞机一停下第一个夺门而出，在偌大的贵阳站像个傻子一样狂奔，结果400m冲刺完后行李等了好久好久才出来，等赶到火车站，火车早走了…大晚上我们在火车门口三脸懵逼…对了，飞机上他还一直在推销他们的飞机模型！！很吵啊！！
    3. 我们发现，在微信服务里购买机票的话，有一个领券中心，点进去向下滑，可以看到东航专区，可以9.9购买那个70优惠券套餐，买机票的时候可以减掉50呢！
      总结： 这一天十分的不开心，计划都被打乱了，没吃到好吃的晚饭，还多花了好多钱😡

Day1—4.22 昆明

  • 计划：

    1. 6点半到昆明，先去找米线吃早餐
    2. 再去斗南湿地（地铁大概40.50min），王冠湿地，预计2.5h；
    3. 接着去斗南花市吃午饭，逛斗南花市，前往高铁站。

    4. 待定午饭地点：傣味云南特色菜；大洋园昭通火腿鸡；嘉华鲜花饼斗南店…

      昆明前往大理：

    5. 昆明-大理高铁站 🚝 17：27-19：38 114r
    6. 前往酒店大理归山民宿，放下行李，步行5min前往大理古城，拍照小摊买裙子，记得多砍价！ 1.8m+1.2m三人间，536/3人

  • 实际：

    1. 10点半到达昆明站，去服务台寄存行李，20r/人，前往斗南花市，2h逛完斗南花市，在斗南花市附近找了家dzdp上评分贼高的甄味斗南吃了个午饭。由于太晒了，吃完饭也快2点了，17：27的高铁票已经提前买好了，剩下的时间去哪个景点都不太来得及，最终决定改签15点的高铁票，提前离开昆明前往大理😅…
    2. 晚上到达大理后，我们幸福的旅程就真正开始了！到归山民宿放下行李就步行去大理古城开始买买买！晚饭是在古镇里面找了家石锅烤肉吃的，边烤肉边有人弹吉他唱歌。逛完后回酒店睡觉！ 烤肉：163/3人

  • 感受：

    1. 斗南花市的花真的很多很便宜，第一次见到按斤称重的玫瑰，二楼好多奇形怪状的多肉，值得去看看，但是邮寄鲜花的快递费太高，接下来还有行程，手拿着花不方便，所以最终我们..空着手…走出了斗南花市…
    2. dzdp还是得选择性相信啊！！！这一天心情不好的最大因素！！就是这家餐厅！！好坑好难吃！！！店确实很高级，据说他们可是楼下维多利亚酒店的早饭供应……可是店里点餐9.9一份小炒牛肉，想在美团买个100-50的优惠券告诉我们只能用一个优惠，点了小炒牛肉就不能用这个美团券了？？您在美团使用规则上咋不说清楚呢？？？这就算了，尊重！最终我们点了美团上的一个套餐，咋说呢，除了小炒牛肉还不错以外都不咋地，xhs上那么多人推荐的那个小麻鸭，我们觉得不如南京烤鸭北京烤鸭…还有两大盆米线！！！一点味道都没有啊！我就是在南京吃云南米线都比这正宗啊！您一个在云南的高级餐厅怎么能把云南米线做成这样啊！要不是后面几天吃到了好吃的米线，我真的就觉得云南米线就是这个没味道的味道了
    3. 除了去云南的票和回南京的机票，别的票真的不要太早的买啊，我们做攻略的时候就把行程间的高铁票也买了，结果一开始的飞机延误让我们在昆明的时间直接减半。不过本来也都说这个时候昆明那个大坝的海鸥也没啥了，所以没太大的遗憾吧。
    4. 这家归山民宿！真的很不错！虽然没有接机服务，但是离古城和苍山都很近，是一个山景房。老板是个小姐姐态度也超好，我们一到就给我们送了个果盘。第二天早上提供早饭，是她们现煮的米线，超好吃！距离大理高铁站18公里，打车30min到达。
    5. 大理古镇里的物价差距真的挺大的，尽量往中间走走再开始买买买！我们最终是在中间一家老爷爷的不豪华但是披肩种类很多的店里买了三条披肩和两条裙子。一条披肩最终25r拿下！白裙子是85r。不过感觉xhs上说的“不用带太多衣服去云南，在大理古城现买”还是不太实际的，毕竟店面都很小，衣服没办法试，而且质量确实不如平时穿的，披肩和白裙子感觉离开云南后就不会再穿了，所以我还是建议大家自己带衣服，在大理古镇买个披肩拍拍照就好~
    6. 大理古镇里的饭店感觉有点少，我们忍着饿走了好远好远想吃一个手抓饭没找到，最终随便找了个烤肉，不过挺好吃的，就是吃到后面开始溅油了，溅到手上贼烫，需要注意点。

Day2—4.23 大理

  • 计划：

    1. 早上民宿阿姨现做米线；
    2. 大理云之蓝海景花园度假民宿接送，前往第二个海景酒店。
    3. 大理云之蓝海景花园度假民宿，1.2m+1.8m 侧海景家庭房一间，1056/三人·两晚
    4. 放下行李后，租自行车，环洱海骑行。

  • 实际：

    1. 10点下楼吃米线，前往大理古城那个菜市场里老爷爷的草包店里购买草包，12点赶回民宿带上行李，坐上来接送的宝马车，前往第二个民宿
    2. 云之蓝民宿就在生态廊道洱海边上，租了民宿门口老爷爷的自行车，沿着洱海前往生态廊道和网红S弯，晚上回民宿边看洱海边看刘亦菲的《去有风的地方》

  • 感受：

    1. 早上的米线很好吃，老爷爷的草包店种类是真的多也是真便宜，买了个容量挺大的水桶草包只要25r，舍友买了三个包均价20也都很好看。
    2. 云之海民宿确实是个海景房，民宿装修的很好看，风景也很美，管家的态度很好。提供早餐，早餐种类十分丰富，我们在这家民宿又吃了两天的米线作为早餐，也很好吃！晚上还有免费的甜汤，喝到了绿豆汤和红豆汤，都非常好喝！
    3. 洱海真的很美，住在洱海边上有一种说不出来的松弛感，这是我离开洱海后再也感受不到的惬意！沿着洱海骑行感觉到处都是S弯，每个地方的风景都很美，反而网红S弯那一段过于商业化，人太多了反而少了那种韵味。
    4. 民宿门口那个租车的老爷爷真的很好！我们最好不小心把锁车的钥匙弄丢了，车锁在一起骑不了了，还开始下雨了。打电话给老爷爷，爷爷立刻赶来安慰我们，让我们先回酒店，他来回骑了好几趟才找到备用的钥匙把锁打开，自己一个人在雨中把我们的两辆车拉回去了，没让我们赔任何钱😭
    5. 生态廊道上，是真的没有任何吃的…除了S弯上那些贵的要死的咖啡店…其他都是民宿提供餐饮这种的，偏贵又不是很有特色，外卖配送费也都贼高。所以我们最后是在民宿旁边那个小摊子买了份晚饭，价格就十几块钱一份，不是很好吃，不过可以填饱肚子。感觉从这天开始，就奠定了我们后来每天午饭晚饭合在一起，每天两顿饭的基调了😂

Day3—4.24 大理

  • 计划：

    1. 没人会开车，所以找司机包车环洱海。

  • 实际：

    1. 民宿预测的7点08分日出，所以我们6点50起床下楼看日出，看完后回酒店补40min觉。
    2. 舍友联系的包车司机，300r一天9h，预约的9点，所以八点下楼吃早饭。9点司机到了，带我们去了廊桥，喜洲古镇，双廊古镇，小普陀，文笔村公路（应我们的要求没停下，在车上看了一眼）和理想邦。最后应我们的要求把我们送到了一家他觉得好吃的手抓饭店。

  • 感受：

    1. 洱海的日出挺美的，只是那天早上的太阳刚好被一片云挡住了，看的不完全，不过还是很惊艳！早上还是冷的，要穿的厚点。在这不得不安利一下我的冲锋衣！真的抗风！且万能！云南6天，它帮我遮阳挡风，甚至爬雪山穿的也是它！让我的箱子多出了一件羽绒服的空间！
    2. 民宿的早饭很好吃，吃的很多很撑，让我们得以撑到下午五点多才开始吃第二顿饭。
    3. 廊桥拍照挺好看的；喜洲古镇挺好逛的，在这我们吃到一家自认为最好吃的鲜花饼，店名看支付宝的收款记录是“花点鲜花饼”，当我们想再买一点寄回家的时候却再也找不到它了，小红书淘宝高德甚至百度都找不到这家店😭；双廊古镇没啥印象了，匆匆的看了一眼；小普陀10min走完了，没啥海鸥了所以没花钱买饼干喂海鸥；理想邦很独特，他的建筑很壮观，我认为值得爬上去看一看。理想邦上有一家泡鲁达，他家的烤肠好好吃！是石板烤的那种，可有味道了；泡鲁达用的不是牛奶是椰奶，里面的料太多了，不是很喜欢。总的来说，古镇逛多了，天气也很晒，这一天很多景点都是走马观花，很多网红打卡地都没去。
    4. 司机带我们去的这家手抓饭店确实挺不错，很多当地人在里面吃，不过我个人不爱吃米饭，所以对手抓饭无感，舍友吃的很开心。店名是金米粒清真傣味手抓饭。对了，云南的这些饭店都可以去美团上搜一搜，基本有优惠套餐！

Day4—4.25 丽江

  • 计划：

    1. 打车去丽江
    2. 前往丽江云山美舍设计师民宿酒店，家庭房 1.8m+1.5m 604/三人·2晚
    3. 拍写真，逛丽江古城

  • 实际：

    1. 前一天晚上在哈喽app上约的顺风车，91.9r/人，10：45出发，大概13：00到达酒店。
    2. 前往丽江古城，无意中遇到鹿森摄影谈了一下后决定在她们家拍写真。298r/人
    3. 大概5点多拍完，在写真店的推荐下在隔壁的阿婆腊排骨吃了午晚饭。
    4. 吃完饭继续逛古城，买了很多小配饰作为伴手礼。

  • 感受

    1. 顺风车的体验感不错，司机大哥已经自驾游3个月，走遍了大半个中国了，接顺风车也只是为了找人分担下路费和高速费，羡慕这种生活~
    2. 民宿体验正常吧，价格摆在这，确实不如之前两个酒店啦，不过床很舒服。
    3. 据说丽江的海拔比大理更高，所以建议先去大理再去丽江，逐步适应，以防高原反应。但是不得不说丽江古城的披肩真的更多更便宜啊！如果确定自己没有高反的话可以先来丽江买买买！
    4. 这家写真的服务态度真不错，第一次拍写真体验感很好，只不过拍完照看照片才发现原来我已经胖了这么多啊😰虽然知道自己疫情期间确实胖了很多，但是没想到胖的这么明显啊！小小的难受了一晚上，励志减肥惹🙀
    5. 腊排骨挺好吃的，配上水性杨花真的超美味，后面就没吃到这么美味的水性杨花了…
    6. 丽江古城里的物价差距也很大，一定要多问几家再买啊！！花了30r买了两个西藏那种手持滚筒挂件以为很便宜了，结果遇到一家老奶奶店，才10r一个！！这家老奶奶店真的很便宜！很多之前问的70一串的手链在这只要20r！奶奶自己也说我们能逛到她这家店挺不容易的，应该是木府1号的位置。还有一家好像是在腊排骨旁边一个大叔的店，5r买了个铃铛挂件10r买了个小风铃，声音很治愈！
    7. 丽江古城晚上有很多音乐酒吧，不过我们太累了就回酒店躺尸了。丽江古城里的水果也很好吃（可能云南水果都好吃，只是我只吃了丽江古城的水果）！小菠萝好甜！我一个不喜欢吃菠萝的人都买了一个！！

Day5—4.26 丽江

  • 计划：

    1. 看日照金山
    2. 爬玉龙雪山（太复杂了，所以没咋研究攻略）
    3. 束河古镇

  • 实际：

    1. 原本在xhs上找了个30r/人带我们去东巴谷看日照金山，再送去玉龙雪山的司机，我们没珍惜；接着哈喽顺风车上有个司机接单了问我们50r/3人带我们去东巴谷看日照金山接不接受，因为平台上顺风车的价格是15r，她涨到了50r，所以我们再次拒绝了…之后再也打不到顺风车了，无奈滴滴预约了第二天早上6点的车，最后54r送我们去东巴谷…
    2. 在楼下的米线店里吃了石锅米线作为早饭 12r
    3. 6点40到达东巴谷等待日照金山，日出大概就在6点50左右
    4. 7点10分滴滴打车去玉龙雪山（所以我们为什么不接受第一个30r包全程的司机呢😭），门票学生票半价50r
    5. 去玉龙雪山购买云杉坪+蓝月谷的电动车套票60r，游玩云杉坪和蓝月谷
    6. 12点电动车回到游客中心，乘坐大巴赶往12点30分的大索道准备上玉龙雪山
    7. 18点多下山，赶往束河古镇
    8. 在束河古镇边上的废墟餐厅吃晚饭
    9. 回宾馆

  • 感受：

    1. 综合来看，确实xhs上包车看日照金山+玉龙雪山最划算！可惜当时我们没有珍惜，滴滴打车花了2倍的钱啊！！（xhs直接搜东巴谷日照金山选择最新就能看到好多当地人开车带你去东巴谷+玉龙雪山，感觉形式和顺风车一样，不过不知道靠不靠谱）
    2. 民宿门口的那家米线也超好吃！！！还不贵！！！
    3. 太阳升起来的时候又刚好被乌云挡住了，没能看到那么完美的日照金山，不过也很震撼了！刚开始我们很疑惑为什么大家都站在那么远的地方，我们特意找了个很近的地方等日出，等太阳出来时才发现大家站的那个地方机位最好啊！！所以尽量往后站，可以拍到日照金山的全貌！
    4. 我们爬玉龙雪山的那天山顶下雪了，虽然前一天晚上看xhs都说不要租衣服，羽绒服足够了，但是我们还是没忍住在山下租了防寒服还买了手套，防寒服25r一天，手套10r一双。事实证明，刚下大缆车在海拔3000多的时候我们就已经裹紧了防寒服，等爬到4680m时已经一把鼻涕一把泪，庆幸自己租了衣服买了手套！
    5. 玉龙雪山大缆车的票是要在前一天晚上8点开始抢，我们研究了xhs上很多抢票攻略，每个人提前填好了三个人的信息，8点三人一起抢，最终只有我成功抢到了。我们选择的是12-12：30那批的大缆车票，因为缆车不能迟到，牦牛坪太远了，所以我们最终选择游玩云杉坪+蓝月湖；
    6. 云杉坪风景还不错，蓝月湖是真的震撼！好蓝好清的水啊！不过穿着防寒服在这两个地方玩是真的热啊
    7. 玉龙雪山真帅啊！我真的很爱这种黑石白雪的感觉了，很幸运我们的高反没有那么严重，到最后100m时确实爬十几级台阶就会累成狗，而且后脑勺有点点疼。我们每个人都在丽江古城买了一篇氧气罐都没用完，还买了红景天每人喝了两瓶，确实有点缓解效果。不过登顶4680m的人确实普遍比较年轻，路上也见到了因为高反呕吐，抱着氧气罐狂吸的人。山上的氧气罐60r一罐，我们11r一罐，所以害怕自己高反的人可以在山下多备点带上去！
    8. 玉龙雪山上4000多米有个亭子卖牦牛肉！超好吃！15r一根！吃完一根神清气爽，感觉头都没那么疼了！
    9. 束河古镇是云南当地人推荐去的一个古镇，确实没那么的商业化，很安静很有古镇的气质，很好玩！！很推荐去！！丽江古城确实人太多了。在束河古镇刚进去的地方吃到了一家的包浆豆腐，好嫩好好吃！后来在丽江古城就没吃到这么好吃的了。
    10. 废墟餐厅！！！我真的要狠狠安利啊！！128r双人餐我们3个人没吃完！每道菜都超好吃啊！！而且环境好美装修好高级狠狠戳中我的审美！我们在室外吃的，还有好看的小姐姐弹吉他唱歌，好惬意！我被种草了他家室外的露营凳，淘宝一搜259r，桌子998r…抱歉打扰了，再看看这个菜的价格，谢谢老板！

Day6—4.27 回程✈

  • 计划

    1. 再逛逛丽江古城
    2. 丽江 🛫 南京（丽江三义 - 南京禄口） 18：25-21：10 812r

  • 实际：

    1. 睡到11点起床，看到东航发的飞机延误到20：00的消息，可以免费改签或者退票（不愧是大航班，拉踩九元！！），本来考虑退票继续去西双版纳，玩两天后从西双版纳回南京，这样一共只比回程的票多了300r左右却可以多玩一个城市，但看完xhs西双版纳病毒的帖子决定不退票直接回南京！
    2. 去丽江古城的云雪丽·木楼餐厅吃早午饭
    3. 逛丽江古城时看到了万古楼，决定去爬一个！学生票17.5r
    4. 17：20离开丽江古城赶飞机👋

  • 感受：

    1. 还是想拉踩九元航空！东航延误可以免费退票欸！！而且空姐的态度都好好！！提供餐饮，我们问有没有奶茶，小姐姐直接去头等舱给我们拿了牛奶，还给我们倒了乌龙茶，自制奶茶get！
    2. 这个木楼餐厅也好吃的！top2的豆腐！惊为天人啊！最后的酱都被我们拿来拌米饭了！还有那个黑松露的什么饭也好好吃！我一个不喜欢吃米饭的人都吃了两碗！
    3. 万古楼确实没啥好看的，但是通往万古楼的路上好多观景的饭店和酒吧，景色都超美！之前都没发现啊！
    4. 结束喽~玩的好开心啊🥰
